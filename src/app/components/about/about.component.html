About

I built this application as an experiment primarily to learn how to use Angular 2. It is based on my real blog sushipilgrim.com The real blog is built in Wordpress and hosted on an Ubuntu VM at Godaddy.com For now the app is populated with dummy reviews. I have production experience with Angular JS and I needed a project to use to learn Angular 2. Since I don't have any plans to duplicate Wordpress's commenting or management features, this app will remain an experiment.

This application is deployed on a Google Compute Engine VM running a MEAN (MongoDB, Express, Angular, Node JS) stack on Debian Linux.

The source for this app is available on Github (app). The source for the RESTful services used by this app is also available on Github (server)
